二、栈Stack

是一个数据集合，只能在一端进行插入或者删除的列表。

后进先出（last-in,first-out,LIFO）。栈底（a[0]）、栈顶a[n]。
进栈（压栈）push：li.append、 出栈pop：li.pop、 取栈顶gettop：li[-1]。
